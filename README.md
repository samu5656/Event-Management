BookMyEvents is a web-based Event Booking System that allows users to explore, add, book, and delete events through an interactive and user-friendly interface. The project is built using HTML, CSS, and JavaScript, focusing on modern UI design and smooth client-side functionality.

Features

View a list of upcoming events with details such as venue, date, price, and registrations

Book events and track the number of registered members

Add new events dynamically

Delete events when required

Search events easily

Responsive and visually appealing UI

Technical Highlights

Uses async/await for handling asynchronous operations

Client-side data handling (no backend or database)

Modular and readable JavaScript code

Deployed using GitHub Pages
